本项目是本人于2018年参加阿里、京东等其他公司电话、现场面试之后总结出来的针对Java面试的知识点，每个点都是被实际的面试中问过的。

有疑问欢迎提 Issues 让我们共同解决，有好的想法想加进来的请提 PR ~

PS：除开知识点，一定要准备好1分钟~3分钟的个人介绍，以及对每个知识点的个人理解（当面试官问你是如何理解多线程的时候，你要知道从定义、来源、实现、问题、优化、应用方面系统性地回答）
